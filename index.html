2. Bank Statement Forensic Analysis System

Problem

Forensic investigations of bank statements were heavily slowed down by varied layout formats, requiring manual data transcription, transaction tagging, and audit trail tracing.

Task

Engineered a bank statement forensic analysis system leveraging Transformer-based AI models to automate statement extraction, transaction classification, data standardization, anomaly detection, and forensic trail generation for investigative and audit purposes.

Process

Designed spatial-layout-aware deep learning pipelines to extract records from scanned sheets. Automated the categorization of statements into structured standard ledgers, tracing transactional paths.

Outcome

Substantially reduced manual review timelines for investigative audits and generated standardized forensic ledger summaries.

3. Material Theft & CV Detection System

Problem

Task

Process

Developed custom computer-vision algorithms using PyTorch and OpenCV. Preprocessed historical procurement logs and sensor scales, applying statistical anomaly thresholds and gradient boosting regression models.

Outcome

Reduced material loss by 50% for client companies, and reduced false-positive flags on security cameras by 20%.
